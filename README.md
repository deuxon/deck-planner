# Deck Planner MVP

## Overview
The Deck Planner MVP project aims to create a simple and user-friendly application to help manage and plan card decks for various games. This document provides a comprehensive guide to using, contributing to, and understanding the project.

## Installation
To install the Deck Planner MVP, please follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/deuxon/deck-planner.git
   ```
2. Navigate to the project directory:
   ```bash
   cd deck-planner
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Usage
To start using the Deck Planner MVP, run:
```bash
npm start
```
This will launch the application in your default web browser.

## Features
- Create and manage card decks.
- Add and remove cards from decks.
- Save and load decks.
- User-friendly interface for easy navigation.
